Métodos CSS

O que são métodos CSS?

Os métodos CSS são técnicas utilizadas para aplicar estilos a elementos HTML. Existem diferentes maneiras de fazer isso, incluindo:

  • Estilos inline
  • Estilos internos
  • Estilos externos

Exemplos de métodos CSS

1. Estilos inline

Os estilos inline são aplicados diretamente no elemento HTML usando o atributo style.

<h1 style="color: blue;">Título Azul</h1>

2. Estilos internos

Os estilos internos são definidos dentro de uma tag <style> no cabeçalho do documento HTML.

<style>
    h1 {
        color: blue;
    }
</style>

3. Estilos externos

Os estilos externos são definidos em um arquivo CSS separado e vinculados ao documento HTML.

<link rel="stylesheet" href="styles.css">
